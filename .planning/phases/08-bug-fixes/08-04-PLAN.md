| route handler → DB (locatarios) | UPDATE com service role (supabaseAdmin) — bypass de RLS intencional para escrita server-side pós-autenticação |

| T-08-07 | Tampering | UPDATE status_convite via service role | mitigate | UPDATE só ocorre APÓS verifyOtp/exchangeCodeForSession bem-sucedido; escopado por usuario_id do user autenticado — não aceita id arbitrário do cliente |
