## Component Inventory

### New Components (inline in `page.js` — plain JS functions, not separate files)

#### `OccupancyBar({ alugadas, total })`

Renders one flex cell per unit. Alugadas cells first (filled), disponíveis cells after (outlined).

```
Layout:    flex row, gap: 3px
Cell size: flex: 1, height: 28px desktop / 20px mobile
Alugadas:  background: var(--color-primary-hover), border: none
Disponíveis: background: var(--surface-hi), border: 1px solid var(--border-3)
```

Canonical: `overview.jsx` lines 48-55.

#### `CashFlowChart({ fluxo, height })`

Six vertical bar columns, one per month in the rolling window. Each column has a ghost (previsto) bar behind a solid (recebido) bar.

```
Container:  display flex, align-items flex-end, gap: 10px, height: {height}px
Per column: flex 1, flex-direction column, align-items center, gap 8px, justify-content flex-end
Bar wrapper: position relative, width 100%, flex 1, align-items flex-end, justify-content center

Ghost bar:  position absolute, bottom 0, width 62%, height {f.previsto}%
            background: var(--secondary), opacity: 0.5

Solid bar:  position relative (in-flow, renders after ghost in DOM order)
            width 62%, height {f.recebido}%
            background: var(--highlight) if f.peak else var(--color-primary-hover)
            boxShadow: "0 0 6px 0 var(--highlight)" if f.peak else "none"
            transformOrigin: "bottom"
            animation: rGrowY var(--dur-base) var(--ease-crisp)
            animationDelay: {i * 60}ms
            animationFillMode: "none"   ← REQUIRED per REFINO-05 (no opacity:0 fill)
            className="chart-bar"       ← for reduced-motion CSS hook

Month label: .r-meta, fontSize: 9px, 3-char lowercase (jan fev mar abr mai jun ...)
```

Canonical: `overview.jsx` lines 28-44. Token correction: use `rGrowY` (vertical scaleY), NOT the existing `rGrow` (horizontal scaleX).

#### New CSS additions to `globals.css` (required)

```css
@keyframes rGrowY {
  from { transform: scaleY(0); }
  to   { transform: scaleY(1); }
}

@media (prefers-reduced-motion: reduce) {
  .chart-bar { animation: none !important; }
}

@media print {
  .chart-bar { animation: none !important; }
}
```

## State Matrix

| State | When | What Renders |
|-------|------|-------------|
| Empty (isEmpty=true) | `edificios.length === 0` | Setup wizard — preserved exactly; OccupancyBar and CashFlowChart NOT rendered |
| Error | Query throws | Error string in `--danger-fg` font-mono; none of the new blocks render |
| No paga parcelas | `getParcelasFluxo()` returns no `status='paga'` rows | All `recebido` bars = 0%; only ghost (previsto) bars show; no peak (gold) bar |
| No parcelas at all | Empty `getParcelasFluxo()` return | Chart renders 6 empty columns (all heights 0%); `maxVal` clamped to 1 |
| 0 unidades (but has edificio) | `unidades.length === 0` | `pctOcupacao = 0`, OccupancyBar renders 0 cells (no-op) — `isEmpty` guard already catches this edge case |
| Vencendo > 0 | `vencendoContratos.length > 0` | Warning banner renders above two-column section |

## Animation Spec

| Animation | Keyframe | Duration | Easing | Fill Mode | Trigger |
|-----------|----------|----------|--------|-----------|---------|
| Page entrance | `rFade` (existing) | `--dur-base` = 220ms | `--ease-crisp` | none | `.r-fade` class on outer wrapper |
| Chart solid bars (desktop + mobile) | `rGrowY` (NEW) | `--dur-base` = 220ms | `--ease-crisp` | **none** (REFINO-05) | inline `animation:` style per bar div |
| Chart bar stagger | delay = `i * 60ms` | — | — | — | 6 bars = 0 / 60 / 120 / 180 / 240 / 300ms |
| prefers-reduced-motion | `.chart-bar { animation: none !important; }` | — | — | — | Media query in globals.css |
| print | `.chart-bar { animation: none !important; }` | — | — | — | `@media print` in globals.css |
