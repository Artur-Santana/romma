# TCC - Romma

> Um software de gestão de aluguéis para o mundo corporativo
> 

# Sobre o Projeto

**Romma** é um sistema de gerenciamento de aluguéis voltado para o mercado corporativo, conectando Proprietários de edifícios e andares com Locatários (empresas) que buscam espaços para suas operações.

O projeto é desenvolvido como Trabalho de Conclusão de Curso (TCC) da graduação em Engenharia da Computação, por um desenvolvedor solo, com foco em velocidade de implementação e qualidade técnica suficiente para impressionar a banca avaliadora.
